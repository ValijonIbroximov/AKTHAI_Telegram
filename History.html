Xavfsizlik tamoyillari va amalga oshirilgan himoya

🔐
Argon2id parol xeshlash

m=65536 KB, t=3, p=2. Har bir xesh noyob tuz bilan. PHC-yaqin format. Vaqt-konstantali taqqoslash (timing attack'dan himoya).

🗝️
JWT HMAC-SHA256 + Redis sessiya

Har bir token noyob JTI bilan. Chiqishda Redis'dan JTI o'chiriladi — token bekor qilish imkoni bor. TTL: 12 soat.

🛡️
RBAC: admin / user rollari

RequireRole("admin") middleware'i /api/v1/admin/* marshrutlarini himoya qiladi. Ochiq ro'yxatdan o'tish butunlay o'chirilgan.

📨
E2EE: Server xabar matnini ko'rmaydi

Signal Protocol (X3DH + Double Ratchet). Server faqat BYTEA ciphertext'ni marshrutlaydi. messages jadvalida ciphertext ko'rsatilmaydi.

⚡
Brute-force himoya (kirish)

5 daqiqada 5 ta urinishdan ortiq: 429 Too Many Requests. 5 marta noto'g'ri parol: hisob 15 daqiqa qulflanadi.

📝
Audit log

Barcha admin amallari (login.success, admin.user.create, admin.user.set_active) IP manzil bilan audit_log jadvaliga yoziladi.

🌐
TLS: Barcha trafik shifrlangan

RSA-4096 o'z-o'zini imzolovchi sertifikat, 10 yillik. WebSocket ham wss:// orqali ishlaydi. SAN: DNS:server.lokal, IP:127.0.0.1.
